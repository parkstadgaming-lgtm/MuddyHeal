--------------------------------------------------------------------------------
1. INTRODUCTION
--------------------------------------------------------------------------------
MuddyHeal is a lightweight, high-performance healing suite designed for 
Paladins and other healers in the 1.12.1 environment. It transforms party 
frames into interactive, smart-casting buttons that optimize mana usage 
automatically.

--------------------------------------------------------------------------------
2. KEY FEATURES
--------------------------------------------------------------------------------
* SMART RANKING: Automatically selects the most efficient spell rank based on 
  the target's missing health to prevent overhealing.
* CLICK-TO-HEAL: Full mouse interactivity (Left, Right, Middle clicks) across 
  the entire frame area.
* RANGE FINDER: Frames fade to 40% opacity when a target is out of healing range.
* DEBUFF ALERTS: Border highlights for Magic (Blue), Poison (Green), and 
  Disease (Brown).
* BLESSING TRACKER: Displays a yellow "B" when a target is missing Paladin 
  Blessings.
* LOW HEALTH ALERTS: Health percentage text turns RED below 30% HP.

--------------------------------------------------------------------------------
3. INSTALLATION
--------------------------------------------------------------------------------
1. Place the 'MuddyHeal' folder into your WoW/Interface/AddOns/ directory.
2. COMPLETELY RESTART World of Warcraft (A /reload will not suffice for 
   the initial XML setup).
3. Log in and type '/mh' to configure your spells.

--------------------------------------------------------------------------------
4. COMMANDS
--------------------------------------------------------------------------------
/mh         - Open the spell configuration menu.
/mh lock    - Locks/unlocks the frame for movement.
/mh scale X - Resizes the UI (e.g., /mh scale 1.2).
