MuddyHeal Patch Notes Log
v1.0.0 — The "Official Release" (Current)

    Initial Launch: Full release of the lightweight Paladin healing suite for WoW 1.12.1 [cite: 2026-02-10].

    Smart Casting Engine: Implemented logic to automatically select healing ranks based on the target's missing health to maximize mana efficiency [cite: 2026-02-10].

    Interactive Frame Redesign: Migrated to a texture-based layering system to fix "dead zones," ensuring the entire 105x35 frame area is clickable [cite: 2026-02-10].

    Range & Visibility: Added a 40-yard range finder that dims frames to 40% opacity when players are out of range or line-of-sight [cite: 2026-02-10].

    Aura Monitoring: Integrated a Blessing Tracker (Yellow "B") for missing Paladin buffs and color-coded borders for dispellable Magic, Poison, and Disease debuffs [cite: 2026-02-10].

    Emergency Indicators: Added health percentage text that dynamically turns red when a unit drops below 30% health [cite: 2026-02-10].

Technical Refinement History

    Texture Fix: Resolved a critical "White Box" issue caused by incompatible modern WoW API calls like SetHorizTile [cite: 2026-02-10].

    Click-Through Fix: Adjusted XML frame strata and set enableMouse="false" on all sub-layers to prevent labels from blocking mouse clicks [cite: 2026-02-10].

    Event Handling: Optimized UNIT_HEALTH and UNIT_MANA event listeners to provide real-time updates without affecting game performance [cite: 2026-02-10].
