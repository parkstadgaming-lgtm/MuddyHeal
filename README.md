## 🚀 Key Features
- **Smart Rank Logic**: Automatically selects the most efficient spell rank based on target health [cite: 2026-02-10].
- **Perfect Click Zones**: Custom texture-layering ensures the entire frame is clickable [cite: 2026-02-10].
- **Aura Tracking**: Blessing indicator and color-coded debuff highlights [cite: 2026-02-10].
- **Range Fading**: Visual feedback when targets are beyond 40 yards [cite: 2026-02-10].

## 🛠 Installation
1. Download the repository as a ZIP.
2. Extract the `MuddyHeal` folder into your `World of Warcraft/Interface/AddOns/` directory.
3. **Important**: Restart WoW completely to initialize XML frames [cite: 2026-02-10].

## ⌨️ Commands
